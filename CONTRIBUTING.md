# Contributing

## Architecture

This is a TypeScript monorepo template providing a flexible directory structure for organizing applications, libraries, and shared configurations.

### Directory Structure

```
ts-template/
├── apps/         # Application projects
├── config/       # Configuration files
├── docs/         # Documentation
├── examples/     # Example code
├── libs/         # Shared libraries
├── patches/      # Patches
├── plugins/      # Plugins
├── references/   # Reference materials
├── scripts/      # Build/run scripts
├── templates/    # Templates
├── scratchpad/   # Scratch workspace
└── xdg/          # [XDG Base Directory Specification](https://specifications.freedesktop.org/basedir/latest/)
```
